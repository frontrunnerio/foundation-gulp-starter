# Foundation Gulp Starter

## Features

- Zurb Foundation
- Tooling from the Google Web Starter Kit
  - Gulp for building
  - Bower for package management
  - Browsersync for browser refreshing and more

## Requirements

- nodejs
- npm
- Ruby 1.9 (for sass compilation)

## Installation

```bash
gem install sass

npm install
npm install -g bower gulp

bower install
```

## Development

```bash
gulp serve
```
